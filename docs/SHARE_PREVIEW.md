# 링크 공유 시 미리보기 설정 (OG 메타 태그)

카카오톡, 페이스북 등에서 링크 공유 시 보이는 썸네일/제목/설명은 **Open Graph(og)** 메타 태그로 설정합니다.

## 설정 위치

`index.html`의 `<head>` 안에 있는 og 메타 태그를 수정하세요.

## 수정할 항목

| 메타 태그 | 설명 | 예시 |
|-----------|------|------|
| `og:title` | 제목 | 이동현 · 송하연 결혼합니다 |
| `og:description` | 설명 문구 | 2026년 5월 9일 오드힐하우스에서 뵙겠습니다 |
| `og:image` | 썸네일 이미지 (절대 URL) | https://도메인/assets/images/hero.webp |
| `og:url` | 사이트 URL | https://도메인/ |

## 주의사항

- **og:image**는 반드시 **절대 URL**이어야 합니다 (https://로 시작)
- 커스텀 도메인 사용 시 `og:image`, `og:url`을 새 도메인으로 변경
- 이미지 권장 크기: 1200×630px (가로형)
- 카카오톡 캐시: 한 번 미리보기가 캐시되면 변경이 늦게 반영될 수 있음
